# Stored Procedure

Here is the document that show all our stored procedure that we are using in our Java project.


##getMapCode
```sql
SELECT id, mapCode
FROM Maps
WHERE Maps.id = idMap
```

##getMapXsize
```sql
SELECT id, sizeX
FROM Maps
WHERE Maps.id = idMap
```

##getMapYsize
```sql
SELECT id, sizeY
FROM Maps
WHERE Maps.id = idMap
```
